  pb.jpg           → background of the Personal Bests section
  gallery/1.jpg    → middle gallery (you can add gallery/2.jpg, gallery/3.jpg, etc.)
  gallery/2.jpg
  gallery/3.jpg
  ...

If a file is missing, the page falls back to a gradient background.
To add more gallery photos, edit GALLERY_COUNT in AboutPage.jsx.
